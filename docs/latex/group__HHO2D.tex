\doxysubsection{Detailed Description}
Class providing helper methods and assemble and solve routines for general 2D H\+HO schemes. 

The \mbox{\hyperlink{classHHO2D}{H\+H\+O2D}} class contains methods applicable to a general 2D H\+HO model. It contains assemble and solve routines usable by any scheme. The global operator and load vector are passed to the model through setters and are then accessible by assemble and solve. The class also contains helper methods for writing H\+HO schemes.
